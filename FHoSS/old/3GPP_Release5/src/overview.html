Documentation for Open IMS Core HSS Implementation.

Overview

This HSS provides a java based implementation of a 3GPP IMS HSS.

Structure

The HSS implementation consists of two parts. The definition of the required interfaces and Java classes for the Cx, Sh and Zh interface and the implementation of these interfaces.

You can find four packages containing the mentioned classes:

  • Cx - The definition of Cx interfaces
  • Sh - The definition of Sh interfaces
  • Zh - The definition of Zh interfaces
  • Hss - The implementation of the interfaces above
